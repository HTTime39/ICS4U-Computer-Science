Job Requirements

Skills Needed

A video game developer needs to be able to take the specifications presented to them by the game designers and write the code that will make those ideas work within the game. They need to be able to use APIs such as a game engine like Source or Unity to help bring the worlds created by the game designers and artists to life in a playable format. In general, problem solving skills and knowledge of a programming language are required to be a game developer. Creativity, passion, and teamwork are also good skills to have as a video game developer.

Required Education

Companies that hire game developers expect job applicants to have a computer science degree, software engineering degree, or similar degree. You might also be able to be hired as a game developer if you have proficient knowledge of programming and the gaming industry. It is very competitive to be hired as a game developer, so having a portfolio of your previous projects can be important to increase your chances of being hired by a game publisher.
